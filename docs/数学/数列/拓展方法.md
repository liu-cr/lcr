#### 二阶线性递推：特征根法

用于求解形如 $a_{n+2}=pa_{n+1}+qa_n$ 的数列 $\{a_n\}$ 通项。

还是用待定系数法，令原式等价于 $a_{n+2}-\lambda a_{n+1}=\mu(a_{n+1}-\lambda a_n)$。还原成左边只有 $a_{n+2}$ 的形式，得到 $a_{n+2}=(\lambda+\mu)a_{n+1}-\lambda\mu a_n$。

发现了什么？$p=\lambda+\mu$，$q=\lambda\mu$，恰好有一个两数之和，一个两数乘积，可以想到韦达定理。也就是说，$\lambda$ 和 $\mu$ 分别为 $x^2-px+q=0$ 的两根。
